# CRM Lead Management System

## Features
- Authentication
- Dashboard
- Lead CRUD
- Notes
- Search
- Filters

## Tech Stack
Frontend:
- React
- Plain CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

## Setup

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start

## Environment Variables

PORT=5000
MONGO_URI=your_uri
JWT_SECRET=your_secret
