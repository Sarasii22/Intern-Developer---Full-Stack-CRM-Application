# CRM Lead Management System

A full-stack Lead Management System built for a small sales team. This project demonstrates strong skills in frontend development, backend API development, database management, authentication, and clean UI/UX design.

---

## Project Overview

This is a **complete CRM application** that allows users to:
- Manage sales leads with full CRUD operations
- Track leads through different sales pipeline stages
- Add and view notes for each lead
- View key performance metrics on a dashboard
- Search and filter leads efficiently

---

## Features

### Frontend
- Clean, modern and responsive UI
- Dashboard with key statistics (Total Leads, Won/Lost, Deal Values)
- Advanced search and multi-filter functionality
- Real-time status updates from the leads table
- Add/Edit leads with form validation
- Detailed lead view with note management

### Backend
- RESTful APIs with Express.js
- JWT-based Authentication
- MongoDB with Mongoose ODM
- Protected routes using middleware
- Proper error handling

### Key Functionalities
- Full CRUD operations on leads
- Notes system (add/view)
- Pipeline status management (New → Won/Lost)
- Priority levels and deal value tracking
- Responsive design for desktop & mobile

---

## Tech Stack

**Frontend:**
- React.js
- React Router DOM
- Axios
- CSS3 (Modern styling with Flex & Grid)

**Backend:**
- Node.js + Express.js
- MongoDB (MongoDB Atlas)
- Mongoose
- JWT (JSON Web Tokens)
